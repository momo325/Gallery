# 🌌 momo325's Image Gallery

这是一个专门为 Markdown 文档、学术笔记及个人项目打造的**云端图床仓库**。

## 🚀 核心特性

- **CDN 加速**：利用 [jsDelivr](https://www.jsdelivr.com/) 提供全球/国内双重加速，确保图片秒开。
- **结构化存储**：按项目类别存储，保持文档引用路径清晰。
- **高可用性**：作为个人研究文档的静态资源库，实现多端同步（台式机 & 笔记本）。

## 🔗 链接引用规范

为了在 Markdown 文档中获得最佳加载速度，建议使用以下 jsDelivr 链接格式：

`https://cdn.jsdelivr.net/gh/momo325/Gallery@main/img/文件名.png`

## 🛠️ 配合工具

- **Editor**: Typedown / VS Code
- **Upload Tool**: PicGo-Core / Manual Upload (Web)
- **Workflow**: 
  1. 截图或导出项目图表。
  2. 手动上传至 `/img` 目录。
  3. 复制链接至 `.md` 文档。

## 📂 目录结构

- `/img`: 存储通用的笔记配图、架构图及算法流程图。
- `/projects`: 存储特定项目（如：双维转台、数字经络）的专项素材。
